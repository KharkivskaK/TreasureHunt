Testing the App

Overview:

The Web App was made as a group assignment by Team-A3.
The TreasureHunt requires that the player can request, answer and skip questions if allowed, certain questions can only be answered if they are at a location provided to the API to be checked.

The plan for testing this app is to introduce methods to test the robustness of the features provided to the users.

Testing Methods:

User Acceptance Testing (UAT): User acceptance testing involves real users testing the application to ensure it can handle required tasks in real-world scenarios, according to specifications.

Unit Testing: Unit testing involves testing individual components or functions of a software application in isolation to verify that each part operates as expected.

Heuristic Evaluation: Heuristic evaluation is a usability inspection method where experts use established heuristics or guidelines to identify usability problems in a user interface design.
